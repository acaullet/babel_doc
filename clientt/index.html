The ClientTcp Class

The class itself


class ClientTcp {
public:
        ClientTcp(std::string);
        ~ClientTcp();
        void onReadyRead();
        void send(std::string message);
        std::string receive();
        int id() const {return _id;}

    protected:
    private:
        std::string _pseudo;
        QTcpSocket _sock;
        int _id;
};

class ClientTcp

This class is the one used in communications between the ClientTcp and the server. Is is contained as an attribute in the Client class.

As a mandatary, everything sent to the server is related to QT.

It has a pseudo, a QTcp socket and also an id.

Constructor

ClientTcp(std::string pseudo);

Arguments:

pseudo : the pseudo of the Tcp client

Usage: This function is the constructor of our class.

OnReadyRead Function

void onReadyRead();
Usage: This function is called when we need to init the read function.

Send Function

void send(std::string message);

Arguments:

message : the message to be send to the server

Usage: This function is called when we need to send a message to the server.

Receive Function

std::string receive();
Return : This function return the message received by the ClientTcp from the server.

Id Function

int id() const;
Return : This function return the id of the ClientTcp in an int.
