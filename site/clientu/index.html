The ClientUdp Class

The class itself

class ClientUdp {
    public:
        ClientUdp(int);
        ~ClientUdp();
        void onReadyRead();
        void send(std::string message);
        std::string receive();

    protected:
    private:
        int _id;
        QUdpSocket _sock;
};

class Client

This class is the one used in communications between the ClientUdp and other(s) ClientUdp. It is contained as an attribute in the Client class.

As a mandatary, everything sent to the other(s) is related to QT.

It has an id and also a QUdp socket.

Constructor

ClientUdp(int id);

Arguments:

id : the id of the Udp client.

Usage: This function is the constructor of our class.
void onReadyRead();
Usage: This function is called when we need to init the read function.

Send Function

void send(std::string message);

Arguments:

message : the message to be send to the other(s) Udp client(s).

Usage: This function is called when we need to send a message to the other(s) Udp client(s).

Receive Function

std::string receive();
Return : This function return the message received by the ClientUdp from an other Udp client.
